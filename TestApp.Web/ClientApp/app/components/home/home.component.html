.NET Core Middleware Demo

Appveyor Build status:

This is an app meant to demonstrate the power and implementation of .NET Core Middleware. It features the following technologies:

  • ASP.NET Core and C# for cross-platform server-side code
  • Angular 2+ and TypeScript for client-side code
  • Webpack for building and bundling client-side resources
  • Bootstrap for layout and styling

Getting started from here is pretty simple:

  1. Navigate to "Route Testing"
  2. Click around on the line items on this page to trigger sample POST requests to the server
  3. Navigate to the "Results" page
  4. Here we can view all of the logged requests since the application fired up and some details on each request
